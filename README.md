## Joker online 

Multiplayer online card game powered by Laravel and Bootstrap.

## Usage:
1. Clone this repository
2. `composer install`
3. `cp .env.example .env`
4. `php artisan migrate`
5. `php artisan queue:listen`
6. `php artisan websockets:serve`
7. Register and Play!
